# Troubleshooting

## Typische Contributor-Probleme

- Veraltete Links nach Umstrukturierung
- Uneinheitliche Dateinamen
- Nicht atomare Commits

## Lösung

- Sidebar und Redirect-Seiten immer mit aktualisieren.
- Namenskonvention strikt anwenden.
