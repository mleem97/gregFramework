# Standalone Mod Template

Use this template to create a new FMF-compatible standalone mod.

## Requirements

- `FrikaModdingFramework.dll` must be installed in game `Mods` folder.
- MelonLoader + generated IL2CPP assemblies.

## Files

- `TemplateMod.cs`: runtime dependency check and minimal update loop.
- `StandaloneModTemplate.csproj`: build configuration.
