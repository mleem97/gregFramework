## FrikaModdingFramework (FrikaMF)

FrikaMF ist ein inoffizielles, community-getriebenes Modding-Framework für das Unity-IL2CPP-Spiel `Data Center`.

> **Wichtiger Hinweis:** Du musst nicht beide Sprachen kennen. Wähle **Rust** oder **C#** — das Framework übernimmt die Kommunikation.

`Inoffiziell` · `Community-driven` · `Keine Affiliation mit WASEKU`

## Technische Kurzfassung

- Runtime-Basis: MelonLoader + HarmonyX + Il2CppInterop
- Dual-Track Modding: direkt in C# oder nativ in Rust via C-ABI/FFI
- Build lokal: `dotnet build` (C#) / `cargo build --release` (Rust)
