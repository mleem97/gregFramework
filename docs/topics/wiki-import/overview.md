# Legacy wiki import

The folder `docs/wiki-import/` holds **Markdown sources** that are synced from the repository’s `.wiki/` tree (see `wiki/scripts/sync-wiki-to-docs.mjs`). Use it for long-form guides that predate the Docusaurus-first layout.

The site sidebar under **Topics → Legacy wiki import** includes **all** pages in this folder (autogenerated), not only the links below.

**Entry points**

- [Home (DE)](/wiki/wiki-import/Home) — audience navigation hub.
- [Architecture](/wiki/wiki-import/Architecture), [Setup](/wiki/wiki-import/Setup)
- [HOOKS](/wiki/wiki-import/HOOKS), [FFI bridge reference](/wiki/wiki-import/FFI-Bridge-Reference)

When editing, prefer updating `.wiki/` and re-running sync, or edit the imported copy if you are only fixing the published site — keep both in mind for drift.
